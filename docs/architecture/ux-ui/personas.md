---

### Persona 1 — Carlos, Corretor de Imoveis (Primario)

- **Idade:** 38 anos
- **Dispositivo principal:** Celular (Samsung Galaxy) + notebook basico
- **Habilidade tecnica:** Baixa a moderada (usa WhatsApp, portais imobiliarios)
- **Cenario de uso:** Fotografa imovel vazio com celular, precisa de imagem decorada em minutos para publicar no ZAP/OLX
- **Necessidade:** Rapidez, simplicidade, resultado pronto para compartilhar via WhatsApp
- **Ref:** PRD §1.3 (Publico Primario), Brief §Target Users

### Persona 2 — Marina, Arquiteta (Terciario)

- **Idade:** 32 anos
- **Dispositivo principal:** MacBook Pro + monitor externo
- **Habilidade tecnica:** Alta (usa SketchUp, AutoCAD, Figma)
- **Cenario de uso:** Tem medidas do ambiente e fotos de referencia de moveis, quer gerar visualizacao para apresentar ao cliente
- **Necessidade:** Inputs multiplos (medidas + fotos de referencia), refinamento fino, qualidade profissional
- **Ref:** PRD §1.3 (Publico Terciario), FR-24, FR-25, FR-26

### Persona 3 — Roberto, Gerente de Marketing de Incorporadora (Secundario)

- **Idade:** 45 anos
- **Dispositivo principal:** Desktop corporativo
- **Habilidade tecnica:** Moderada
- **Cenario de uso:** Precisa gerar staging de 5 tipologias de apartamento para lancamento, com variacao de estilos
- **Necessidade:** Troca rapida de estilos, volume, alta resolucao para material impresso
- **Ref:** PRD §1.3 (Publico Secundario), FR-03
