# Casa Arquitetura — Guia Pratico

> Comece a usar a plataforma em 5 minutos.

---

## 1. Criar conta

Acesse a plataforma e cadastre-se com Google (recomendado) ou email/senha.

## 2. Enviar foto do ambiente

Clique em **Novo Projeto** e envie a foto.

**Dicas para melhor resultado:**
- Fotografe de frente, mostrando o maximo do ambiente
- Use luz natural
- Resolucao minima: 1024x1024px
- Formatos: JPG, PNG, WEBP

## 3. Escolher estilo

Selecione um dos 10 estilos:

| Estilo | Ideal para |
|--------|-----------|
| Moderno | Apartamentos urbanos |
| Industrial | Lofts, estudios |
| Minimalista | Espacos pequenos |
| Classico | Casas amplas |
| Escandinavo | Qualquer ambiente |
| Rustico | Casas de campo |
| Tropical | Varandas, areas externas |
| Contemporaneo | Versatil |
| Boho | Quartos, salas intimas |
| Luxo | Imoveis de alto padrao |

## 4. Gerar e refinar

Clique em **Gerar** e aguarde 10-30 segundos.

### Refinar pelo chat

Apos gerar, peca ajustes em linguagem natural:

```
"Troca o sofa por um maior"
"Muda o piso para madeira clara"
"Adiciona mais plantas"
"Deixa a iluminacao mais quente"
```

### Editar elementos

- Selecione um elemento na imagem (parede, piso, movel)
- Troque material, cor ou textura
- Remova objetos indesejados

## 5. Compartilhar

Clique em **Compartilhar** para gerar um link publico com:
- Foto original (antes)
- Render decorado (depois)
- Slider de comparacao interativo

Nao precisa de conta para visualizar.

---

## Diagnostico Gratuito

Ferramenta aberta (sem cadastro):

1. Acesse **Diagnostico** na pagina inicial
2. Envie a foto do ambiente
3. Receba analise com pontos de melhoria e sugestao de estilos

## Casos de uso rapido

| Perfil | Fluxo |
|--------|-------|
| **Corretor** | Foto do imovel vazio → render decorado → link no anuncio |
| **Arquiteto** | Foto do ambiente atual → render no estilo proposto → aprovacao do cliente |
| **Incorporadora** | Foto do espaco cru → render premium → material de venda |
| **Proprietario** | Foto do comodo → testar estilos → decidir antes de reformar |
